How it works

1

Install

One command. Zero dependencies. Pure Python.

2

Init

codepet init installs a git hook in your repo.

3

Code

Every commit earns XP. Bug fixes, tests, refactors — each grows different stats.

4

Evolve

Watch your pet level up through 5 evolution stages: Egg → Legend.

pip install codepet

🎲 Deterministic Hatching

Your pet is generated from your user ID. Same person, same pet — across machines, forever.

📈 Real Stat Growth

DEBUGGING, PATIENCE, CHAOS, WISDOM, SNARK — stats grow based on what you actually code.

🏆 15 Achievements

First Blood, Night Owl, Bug Slayer, Centurion, and more. Unlock badges as you code.

🔥 Streak System

Commit daily to build streaks. 3-day, 7-day, 30-day milestones with bonus XP.

🪶 18KB, Zero Deps

Pure Python. No network calls. No telemetry. Works offline. Installs in 1 second.

🎯 Smart Classification

Auto-detects commit type from conventional commits or message heuristics. No config needed.
